Descripción
Salón de Belleza JLOes una microempresa con el giro de salón de belleza llamado. La propietaria del negocio opera como la estilista principal para los servicios relacionados con la estética del cabello, sin embargo, también ofrece la oportunidad a otros estilistas de prestar otro tipo de servicios como los relacionados a la estética de uñas y pestañas, entre otros.

Problema identificado
Salón de Belleza JLO, es un negocio que hoy en día lleva todos sus procesos de operación de forma manual, por lo que con éste proyecto se busca llevar el negocio a procesos más automatizados y que brinden una ventaja competitiva respecto a los negocios con el mismo giro en su localidad y llevar un registro de las operaciones que se realizan cotidianamente, lo cual permitirá a la empresa tener mayor visibilidad sobre el estatus de negocio y una mejor toma de decisiones.

Solución 
La solución será una aplicación web con la cual las necesidades que se buscan solventar son:

1.	Asignar de turnos: Esta funcionalidad permitirá asignar a cada cliente que llegue de forma directa al negocio un turno de atención, de acuerdo con el servicio y estilista de su elección.
2.	Gestionar turnos: El administrador, podrá visualizar el historial de los turnos asignados durante el día y gestionarlos asignando a cada uno los atributo para su gestión.
3.	Agendar cita: Permitirá reservar un espacio de tiempo específico a un cliente para ser atendido para el servicio y estilista de su elección.
4.	Gestionar citas: Permite cambiar estatus a citas agendadas.
5.	Gestionar estilistas: Permitirá al administrador gestionar a los estilistas externos, con los movimientos de alta y baja y modificación de estos.
6.	Reporte de ingresos: Permitirá tener visibilidad sobre los ingresos diarios de acuerdo con los turnos y citas registrados durante el día.

Arquitectura

La aplicación que se está desarrollando con la siguiente tecnología:

1.	Apache Netbeans IDE 18
2.	Versión de Java 8
3.	Spring MVC Framework
4.	Servidor Tomcat 8.5
5.	Se crearon archivos jsp y son controlados con servlets con método POST usando notaciones.
